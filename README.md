# Excuse Generator

Static web app that generates random excuses using JavaScript.

## Live Deployment (GitHub Pages)

This repository is configured to deploy automatically with GitHub Actions.

After every push to `main`, GitHub builds and publishes the site to GitHub Pages.

### One-time setup in GitHub

1. Open your repository on GitHub.
2. Go to **Settings** > **Pages**.
3. In **Build and deployment**, select **Source: GitHub Actions**.
4. Save.

### Your public URL

Your site will be available at:

`https://gallagit.github.io/excuses-generator/`

If the URL does not load immediately, wait 1-2 minutes after the workflow finishes.

## Run locally

Open `index.html` in your browser.
